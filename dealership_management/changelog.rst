Changelog
=========

**1.0.1 Version (23 APR 2025)**
------------------
- Code optimization for improved performance and maintainability.
- Added a detailed README file for module overview.

**1.0.0 Version (14 OCT 2024)**
------------------
- Initial release of **dealership_management** module.
- Facilitates complete dealership management within Odoo.
- Supports multi-website, multi-currency, and multi-language environments.
- Dealers can register from the front-end, with admin-controlled approval or rejection.
- Tracks the history of admin-dealer queries for transparency and record-keeping.
- Applicants receive stage-wise notifications throughout the dealership process.
- Robust Contract Management system integrated into the workflow.
- Admin can publish categorized FAQs for customers and dealership applicants.
- Enhanced Dealer and Admin Dashboards for real-time tracking of dealership data.
- Customers can locate dealers worldwide via an interactive map, with support for location marking.
